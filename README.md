# C/C++ Learning Journey

Practice exercises and small projects while learning C and C++ from scratch.

Following:
- **C Programming: A Modern Approach** (K.N. King) for C
- **C++ Primer Plus** (Stephen Prata) for C++

## Projects

### C
- `calculator_functions.c` — basic calculator using functions and switch
- `guesssingNumber_game.c` — number guessing game
- `guessingnumber_withLogin.c` — number guessing game with login system
- `addition_game.c` — simple addition practice game
- `matriz.c` — multidimensional arrays (matrices)

### C++ (`/cpp`)
- `login.cpp` — login system using `std::string` and functions
- `vector_practice.cpp` — practice reading user input into a `std::vector`

## Goals
- Understand the logic behind every line of code, not just copy-paste solutions
- Practice one concept at a time, building on the previous one
- Currently learning: **Pointers** (C, Chapter 11)...
